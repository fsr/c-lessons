Short description

The course gives an introduction to the basic ideas of imperative programming languages. Participants will be able to write small C programs which can solve mathematic tasks. Computer workplaces are offered but bringing an own laptop is recommended.

Intended audience

Everyone.

Knowledge requirements

Students should be able to work with plain text editors. Experience in programming is not needed but can be useful in the beginning of the course.

Time requirements

The course will take place for 6 weeks. There are two lessons (4 SWS) per week.
